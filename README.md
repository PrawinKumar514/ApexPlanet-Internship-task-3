# 👥 User Management System

A modern PHP & MySQL based User Management System developed as part of the ApexPlanet Internship Program.

## 🚀 Live Features

### 🔐 Authentication System
- User Registration
- User Login
- User Logout
- Password Hashing
- Session Management
- Form Validation

### 👤 User Profile Management
- View Profile
- Update Profile
- Change Name & Email
- Upload Profile Picture
- Default Profile Image Support

### 🛠️ Admin Features
- View All Users
- Add New Users
- Edit Existing Users
- Delete Users
- Search Users
- Role Management (Admin/User)

### 📊 Dashboard
- Personalized Dashboard
- User Role Display
- Account Information
- Quick Actions Panel

### 🎨 UI Features
- Responsive Design
- Bootstrap 5 Interface
- Modern Dashboard Layout
- Gradient Authentication Pages
- Interactive Alerts
- Clean User Experience

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/PrawinKumar514/ApexPlanet-Internship-task-3.git
```

### 2️⃣ Move Project

Copy project folder into:

```text
xampp/htdocs/
```

### 3️⃣ Create Database

Open:

```text
http://localhost/phpmyadmin
```

Create database:

```text
user_management_system
```

### 4️⃣ Import Database

Import:

```text
database/database.sql
```

### 5️⃣ Start Services

Start:

- Apache
- MySQL

from XAMPP Control Panel.

### 6️⃣ Run Project

```text
http://localhost/User-Management-System
```
